04장. 인터랙티브 웹을 위한 css 다루기

04-1. 사용자 행동에 반응하는 전환 효과 만들기


속성명 속성값 설명
transition-property 속성명 전환 효과를 줄 css 속성명을 지정. 속성명은 여러개 지정 가능하고 all 입력 시 전체 속성 사용
transition-duration 지속 시간(초) 전환 효과가 발생할 때 지속할 시간
transition-delay 지연 시간(초) 전환 효과가 발생할 때 지연할 시간. 이 효과 사용 시 지연 시간 이후 전환 효과 나타남
transition-timing-function 가속도 전환 효과의 가속도 나타냄 - linear : 등속 효과 - ease : 가속 효과 - ease-in : 모션 시작 시 가속 - ease-out : 모션 종료 시 가속 - ease-in-out : 모션 시작, 종료 시 가속 - cubic-bezier : 사용자가 지정한 가속 효과 적용 (속성값 사이트)
transition 속성명/전환시간/가속도/지연시간 모든 전환 효과 관련 속성값 한번에 축약
