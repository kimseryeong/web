04-3. 자동으로 움직이는 애니메이션 효과 만들기


속성명 속성값 설명
@keyframes 애니메이션 세트 지정 애니메이션의 시작과 끝 등록하여 사용자 모션 등록. 시작:0%, 끝:100%, 중간 지점 여러 개 추가 가능.
단계별로 동작 방식 지정해서 특정 이름 붙여 키프레임 등로 가능
animation-name 이름 키프레임으로 등록한 모션의 이름 호출
animation-duration 지연 시간(초) 키프레임 모션 한 세트를 얼마나 동작할지 초단위로 등록
animation-timing-fuction 가속도 키프레임 모션 실행할 때 가속도 설정 (속성값 사이트)
animation-iteration-count 횟수 키프레임 모션 한 세트가 몇 번 동작하는지 횟수를 숫자로 설정. infinite : 무한
animation-delay 지연 시간(초 단위) 키프레임 모션을 실행할 때 지연 시간 지정
animation-play-state running/paused 키프레임 모션을 실행할 때 동작 상태 지정. running : 모션 그대로, paused : 일시정지
animation 이름/진행시간/가속도/지연시간/반복횟수 애니메이션 속성 축약하여 작성
